# myhttpd
## A simple HTTP Server based on Libuv.

* **Platform**: linux
* **Language**: C++
* **Operating system**: ubuntu 16.04 LTS 
* **gcc version**: 5.4.0
* **Libuv version**: 1.23.0 (Stable)
* **Other library**: http_parser
* **Use CMake to compile**<br>
* Install **libuv** and **http_parser** before complie myhttpd.
